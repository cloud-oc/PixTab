## 📖 개요

**PixTab**은 새 탭 페이지에 Pixiv 일러스트를 감각적으로 보여주는 가벼운 브라우저 확장 프로그램입니다. 사용자 설정, 키워드 검색, 다국어를 지원하며 Chromium 및 Firefox 기반 브라우저에서 완벽하게 작동합니다. (✿◡‿◡)

## ✨ 주요 기능

- 🎨 **아름다운 일러스트** — 새 탭을 열 때마다 엄선된 Pixiv 작품을 감상하세요.

<!-- (Skipping lines) -->


- 📊 **다양한 랭킹** — 일간, 주간, 월간, 신인, 오리지널, 인기 순위 등 다양한 랭킹 지원!
- 🔍 **키워드 검색** — AND, OR, NOT 조합을 통해 원하는 태그의 작품을 정확하게 검색합니다.
- 🔖 **북마크 필터** — 최소/최대 북마크 수를 설정하여 인기 작품만 골라보세요.
- 👤 **작가 지정** — 작가 ID로 필터링하여 특정 크리에이터의 작품만 표시합니다.
- ⭐ **팔로우 중인 작가** — Pixiv에서 팔로우하는 작가의 최신 작품을 표시합니다.
- 💝 **북마크 작품** — Pixiv에서 북마크한 일러스트를 탐색합니다.
- ✨ **추천 작품** — Pixiv가 추천하는 개인화된 콘텐츠를 표시합니다.
- 🔐 **로그인 지원** — Pixiv 계정에 로그인하여 개인화 기능을 이용할 수 있습니다.
- 🖼️ **타입 필터** — 일러스트, 만화, 우고이라(Ugoira) 선택 및 AI 생성 작품 숨김 기능 지원.
- 📏 **화질 관리** — 최소 너비/높이를 설정하여 고해상도 이미지만 볼 수 있습니다.
- 🛠️ **커스터마이징** — 이미지 크기, 정렬 방식, 타일 모드를 취향대로 조절하세요.
- 🌓 **테마 동기화** — 시스템 시간에 따라 라이트/다크 모드가 자동으로 전환됩니다.
- 🌍 **다국어 지원** — 영어, 중국어(간체/번체), 일본어, 한국어, 러시아어를 지원합니다.
- 🌐 **리버스 프록시** — 사용자 정의 프록시 URL을 설정하여 제한된 지역에서 Pixiv에 접근 가능.
- 🛡️ **개인정보 보호** — 모든 설정은 로컬에 저장되며 외부 서버로 데이터를 전송하지 않습니다.

## 📦 오프라인 설치

> **참고**: 이 확장 프로그램은 Chrome 웹 스토어에 등록되어 있지 않습니다. Chrome 사용자는 아래 단계에 따라 수동으로 설치해주세요.

<details>
<summary><strong>📥 Chromium 기반 브라우저 (Chrome, Edge, Brave...)</strong></summary>

1. 이 저장소를 복제하거나 다운로드합니다.
2. 브라우저 주소창에 `chrome://extensions`를 입력하여 이동합니다.
3. 우측 상단의 **개발자 모드** (Developer mode)를 활성화합니다.
4. **방법 A — 패키지 파일(ZIP) 설치 (추천):**
    - `dist/` 폴더에 있는 릴리즈 패키지(예: `dist/pixtab-0.9-chrome.zip`)를 다운로드한 경우, `.zip` 파일을 확장 프로그램 관리 페이지로 드래그 앤 드롭하세요.
    - 드래그가 작동하지 않으면 ZIP을 압축 해제한 후, **압축해제된 확장 프로그램을 로드합니다** (Load unpacked) 버튼을 클릭하고 폴더를 선택하세요.
5. **방법 B — 소스 코드 설치 (개발자용):**
    - **압축해제된 확장 프로그램을 로드합니다**를 클릭하고 프로젝트 루트 폴더를 선택합니다.
6. 새 탭을 열고 PixTab을 즐기세요! ✨
</details>

<details>
<summary><strong>🦊 Firefox 기반 브라우저 (140+)</strong></summary>

1. 이 저장소를 복제하거나 다운로드합니다.
2. Firefox 140 이상 버전에서 `about:debugging#/runtime/this-firefox`로 이동합니다.
3. **임시 부가 기능 로드...** (Load Temporary Add-on)를 클릭합니다.
4. **방법 A — XPI 패키지 설치 (테스트용):**
    - `dist/*.xpi` 파일(예: `dist/pixtab-0.9-firefox.xpi`)을 다운로드했다면, `about:addons` 페이지에서 톱니바퀴 아이콘 ⚙️ → **파일에서 부가 기능 설치...**를 선택하거나 `.xpi` 파일을 페이지로 드래그하세요.
5. **방법 B — 임시 로드 (개발자용):**
    - **임시 부가 기능 로드...**를 클릭하고 프로젝트 폴더 내의 `manifest.json` 파일을 선택합니다.
6. 새 탭을 열고 PixTab을 즐기세요! ✨

> *팁*: Firefox에서 임시로 로드된 확장 프로그램은 브라우저를 재시작하면 제거됩니다.
</details>

## 🎐 네트워크 요구사항

이 확장 프로그램은 Pixiv(`pixiv.net` 및 `pximg.net`)에 접속하여 데이터를 가져옵니다.
> 만약 로딩 애니메이션이 멈추지 않는다면 Pixiv에 접속할 수 없는 네트워크 환경일 가능성이 높습니다. 인터넷 연결이나 프록시 설정을 확인해주세요. 이 확장 프로그램은 프록시 기능을 **제공하지 않습니다**.
